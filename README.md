# Baltrus Lab Tutorials

Welcome to the Baltrus Lab tutorials repository. This repository contains step-by-step guides and resources for common tools, workflows, and analyses used in the lab.

---

## Contents

Tutorials are organized into individual folders. Each folder contains a dedicated README and any relevant files for that topic.

| Folder | Description |
|---|---|
| [Blast-Tutorial](https://github.com/surtlab/Baltrus_Lab_Tutorials/tree/main/Blast-Tutorial) | Installation and usage guide for NCBI BLAST+ on Windows and macOS |

---

## Getting Started

If you are new to the lab or to a particular tool, start with the README inside each tutorial folder — it will walk you through installation, setup, and basic usage.
